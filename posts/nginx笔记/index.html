mac下nginx的下载安装和使用#

使用brew安装#

  1. 安装
brew install nginx
  1. 检查nginx配置信息
brew info nginx

安装的文件路径#

配置文件：/opt/homebrew/etc/nginx/nginx.conf

默认服务器地址：/opt/homebrew/var/www

安装路径：/opt/homebrew/Cellar/nginx/1.23.3

配置文件nginx.conf#

# 进程数（与核相关）
worker_processes  1;
# pid
pid /opt/homebrew/etc/nginx/logs/nginx.pid;
# 允许连接数
events {
    worker_connections  1024;
}
# http配置部分
http {
		# 引入配置文件与返回类型
    include       mime.types;
    default_type  application/octet-stream;
    # 超时时间
    keepalive_timeout  65;
		# server部分
    server {
    		# 端口号
        listen       8080;
        # 服务器地址
        server_name  0.0.0.0;
        # 路由
        location / {
        		# 项目根目录
            root   /Users/kimi/Documents/GitHub/generate-visualization/backend/dist;
            # index
            index  index.html index.htm;
        }
        # ——————————————————————————也可写成————————————————————————————————————
        #root   /Users/kimi/Documents/GitHub/generate-visualization/backend/;
        #location /dist {
        #    root   /Users/kimi/Documents/GitHub/generate-visualization/backend/dist;
        #    index  index.html index.htm;
        #}
        # ————————————————————————————————————————————————————————————————————
        # 50x的错误页面
        error_page   500 502 503 504  /50x.html;
        location = /50x.html {
            root   html;
        }
    }
    include servers/*;
}

修改配置文件后，要使用命令指定此配置文件来启动nginx，才有效

相关命令#

查看nginx服务是否启动

ps -ef|grep nginx

验证一下配置文件是否正确

sudo /opt/homebrew/Cellar/nginx/1.23.3/bin/nginx -t -c /opt/homebrew/etc/nginx/nginx.conf

解决nginx启动报错问题

pid问题

# nginx.conf
pid /opt/homebrew/etc/nginx/logs/nginx.pid;

启动及关闭命令#

  1. 按照配置文件启动
nginx -c /opt/homebrew/etc/nginx/nginx.conf
  1. 立即停止
nginx -s stop
  1. 重启
nginx -s reload

Django + Nginx#

Django代理runserver#

  1. 在nginx配置文件中添加
server {
    listen       8080;
    server_name  0.0.0.0;
		location /{
			proxy_pass http://127.0.0.1:8000;
		}
}
  1. 启动django项目，8000端口，0.0.0.0/127.0.0.1
  2. 可以通过127.0.0.1:8080的nginx反向代理访问django

uwsgi#

下载安装#

  1. mac下载安装uwsgi
conda install --channel https://conda.anaconda.org/conda-forge uwsgi

安装文件地址：/Users/kimi/opt/anaconda3/bin/uwsgi

  1. 测试uwsgi是否有效

新建 test.py 文件，内容如下：

def application(env, start_response):
    start_response('200 OK', [('Content-Type','text/html')])
    return [b"Hello World"]

运行：

uwsgi --http :8001 --wsgi-file test.py

运行#

杀死进程

sudo pkill -f uwsgi -9
